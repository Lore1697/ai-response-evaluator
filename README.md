# AI Response Quality & Risk Evaluator

A Python-based rule system that evaluates AI-generated responses for potential quality and safety risks.

## What it does

The tool reads prompts and AI responses from a CSV file and detects issues such as:

- Overconfidence
- Medical risk
- Missing follow-up questions
- Budget mismatch

Then it generates:

- Risk score
- Risk level
- Detected issues
- Suggested improvements

## Technologies

- Python
- pandas
- rule-based text analysis

## Why I built it

This project was created to explore AI output evaluation, risk analysis, and quality assessment in language-based systems.
